# Funding

如果您觉得这个项目对您有帮助，可以考虑通过以下方式支持我们：

## 支持方式

### GitHub Sponsors
如果您想定期支持项目开发，可以通过 GitHub Sponsors 赞助我们。

### 一次性捐赠
您也可以通过以下方式进行一次性捐赠：

- [Buy Me a Coffee](https://www.buymeacoffee.com/)
- [Ko-fi](https://ko-fi.com/)
- [PayPal](https://www.paypal.com/)

### 其他支持方式

- ⭐ 给项目点一个 Star
- 🐛 报告 Bug 和问题
- 💡 提出新功能建议
- 📝 改进文档
- 🔀 提交 Pull Request
- 📢 分享给其他人

## 致谢

感谢所有支持者和贡献者！您的支持是项目持续发展的动力。
